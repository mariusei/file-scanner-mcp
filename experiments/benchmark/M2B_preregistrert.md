# M2b forhåndsregistrerte forventninger (skrevet FØR første episode)

1. Konsept-/strukturoppgaver: scantool-agenter minst like korrekte som
   grep-agenter, med >=2x færre loggede output-tokens
2. Literal-oppgaven (T4): ~uavgjort
3. Oversikt (T5): mulig grep-seier (målt i harnessen)
4. Åpen hypotese: ekte agenter REFORMULERER søk — kan hjelpe begge sider,
   effekt ukjent
5. Modellvalg: haiku-subagenter, bevisst — verktøyverdi skal vise seg
   sterkest for svakere modeller. Risiko: instruksjonsbrudd; mitigeres ved
   logg-verifisering (fakta i svar MÅ finnes i loggen, ellers ugyldig)
6. Null-utfall er gyldig: hvis grep-agenter matcher scantool-agenter på
   korrekthet OG tokens, er informasjonslevering-marginen ikke
   oppgaverelevant — det ville svekke v1.0-påstanden
